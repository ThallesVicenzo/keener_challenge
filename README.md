## How to Run the Project

1. **Clone the repository:**
   ```bash
   gh repo clone ThallesVicenzo/keener_challenge
   ```
   
2. **Install dependencies:**
   ```bash
   flutter pub get
   ```
   
3. **Run the project:**
   ```bash
   flutter run
   ```
   
4. **APK:**
   I have also provided an APK for testing on any ANDROID device.
